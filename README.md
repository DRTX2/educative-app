### 📌 Nombre del proyecto

Prototipo PWA para apoyo educativo en comunidades rurales

---

### 🎯 Objetivo

Desarrollar un prototipo funcional que permita a estudiantes gestionar sus actividades escolares, acceder a contenido interactivo y utilizar la aplicación incluso sin conexión a internet.

---

### 👥 Usuarios objetivo

* Estudiantes de comunidades rurales
* Posible participación de docentes (futuras versiones)

---

### ⚙️ Funcionalidades clave (MVP - 1 semana)

#### 1. Gestión básica de tareas

* Crear y visualizar tareas escolares
* Marcar tareas como completadas

#### 2. Modo offline básico

* Acceso a la aplicación sin conexión
* Persistencia de datos básicos en el dispositivo

#### 3. Contenido interactivo simple

* Actividades tipo preguntas o ejercicios rápidos
* Retroalimentación inmediata al responder

#### 4. Sistema de retroalimentación

* Encuestas rápidas dentro de la app
* Campo opcional de comentarios

---

### 🔄 Flujo básico de uso

1. El estudiante abre la aplicación
2. Consulta o registra tareas
3. Realiza una actividad interactiva
4. Recibe retroalimentación inmediata
5. Puede dejar opinión sobre su experiencia

---

### 📊 Validación del prototipo

El prototipo será evaluado en base a:

* Facilidad de uso
* Comprensión del contenido
* Utilidad percibida por el estudiante
* Retroalimentación recopilada dentro de la app

---

### 🧱 Alcance de esta versión

Incluye:

* Funcionalidades esenciales para validación
* Interfaz simple enfocada en móvil

No incluye (por ahora):

* Sistema avanzado de usuarios
* Gamificación completa
* Integración con backend complejo

---

### 🛠️ Enfoque técnico (resumen)

El prototipo será desarrollado como una **PWA (Progressive Web App)** para asegurar compatibilidad con dispositivos móviles y de escritorio, incluyendo funcionamiento offline básico.
